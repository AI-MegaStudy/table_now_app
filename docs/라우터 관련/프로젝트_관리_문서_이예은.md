## 프로젝트 개요

### 프로젝트 정보

- **프로젝트명**: Table Now
- **앱 유형**: 선결제 방식의 식당 예약 앱
- **예시 도메인**: 카레 프렌차이즈 (코코이찌방야, 아비꼬, 토모토 카레 등)
- **주요 특징**: 예약 시 메뉴를 미리 주문하고 결제하는 선결제 시스템

### 기술 스택

- **프론트엔드**: Flutter (Dart)
- **백엔드**: FastAPI (Python)
- **데이터베이스**: MySQL 8.0
- **인증**: Firebase Auth, Google Sign-In
- **결제**: Toss Payment

## 중요 사항

### 선결제 시스템

- 예약 시 메뉴를 미리 주문하고 결제를 완료해야 예약이 확정됨
- 결제 완료 후 예약이 확정되는 플로우 구현 필요
- `reserve` 테이블의 `payment_status` 필드로 결제 상태 관리

### 날씨 정보 연동

- 예약 시점의 날씨 정보를 저장하여 예약 내역에 포함
- 날씨 API 연동 필요 (OpenWeatherMap 또는 기상청 API)
- `weather` 테이블에 날씨 정보 저장

### 소셜 로그인

- Google Sign-In 연동
- Firebase Auth를 통한 소셜 로그인 처리
- 소셜 로그인 시 사용자 정보를 `customer` 테이블에 저장

### 데이터베이스 스키마

- 현재 스키마는 `fastapi/mysql/table_now_db_init_v1.sql`에 정의됨
- 시드 데이터는 `fastapi/mysql/table_now_db_seed_v2.sql`에 포함됨
- 카레 프렌차이즈 예시 데이터 포함 (코코이찌방야, 아비꼬, 토모토 카레)
